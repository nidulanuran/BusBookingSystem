# Bus Booking System

A full-stack bus booking application that allows users to search, filter, and book bus tickets. The system supports multiple user roles including passengers, conductors, and administrators.It makes effective communication among users and reduce confusions.



## ✨ Features

### Passenger Features
- User registration and authentication
- Search and filter available buses by date, route, and price
- View detailed bus information with seat availability
- Book tickets for trips
- View booking history
- Cancel bookings

### Conductor Features
- View assigned bus information
- Manage passenger list for trips
- Update trip status
- Monitor seat availability

### Admin Features
- Manage bus fleet (add, edit, delete buses)
- View all bookings and trip schedules
- User management
- System analytics and reports
- Configure bus routes and schedules

## 🛠️ Tech Stack

### Backend
- **Framework:** Spring Boot 3.5.6
- **Language:** Java 17
- **Build Tool:** Maven
- **Database:** PostgreSQL
- **ORM:** Spring Data JPA
- **Additional Libraries:**
  - Lombok (code generation)
  - Spring Validation
  - Spring DevTools

### Frontend
- **Framework:** React 19.2.0
- **Build Tool:** Create React App (CRA)
- **Router:** React Router v7.9.6
- **HTTP Client:** Axios v1.13.2
- **Styling:** CSS3

### Database
- PostgreSQL (relational database for data persistence)
